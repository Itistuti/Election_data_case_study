# Election Data Case Study
This project performs exploratory data analysis on election candidate data using Python. It focuses on understanding candidate demographics, party representation, constituency-level winners, and electoral competitiveness through vote margins.

## Dataset
The dataset used in this project is available at:
https://drive.google.com/drive/folders/1w5hEC-qa5HZJiTXY8rbu1eisYqA51WJu?usp=sharing

### Files Used:
- election_data.csv
- district_data.csv
- province_data.csv

## Objectives
The main goals of this analysis are:
- Analyze candidate demographics (gender and age distribution)
- Identify top-performing candidates and political parties
- Examine voting patterns across constituencies and provinces
- Measure electoral competitiveness using vote margins

## Analysis Performed
- Count of male and female candidates
- Number of Gen Z candidates (age 14–29)
- Top 3 candidates by vote count
- Top 10 political parties by number of candidates
- Constituency-wise winners and female winner percentage
- Party-wise constituency wins
- Province with highest total votes
- Winner and runner-up identification per constituency
- Vote margin calculation between top two candidates
- Top 10 most competitive constituencies (lowest margins)

## Tools & Technologies
- Python 3
- Pandas
- Matplotlib
- Jupyter Notebook

## Key Insights
* Voting patterns vary significantly across provinces and constituencies
* Some constituencies are highly competitive with very small vote margins
* Party representation differs widely in candidate distribution
* Gender representation and youth participation can be clearly observed in the dataset

## Skills Demonstrated
* Data cleaning and preprocessing
* Exploratory data analysis (EDA)
* GroupBy operations and aggregation in Pandas
* Sorting, ranking, and filtering data
* Deriving insights from structured datasets
* Basic data visualization using Matplotlib
